# Driveable-Couch
Engineering Club Year 1 Second Semester Project

A few peers and I built a driveable couch as part of our engineering club. We took a base go kart, fixed the pullstart, extended its components, added steering, brakes, seatbelts, and built a custom frame to mount to a couch. It weighs around 300lbs and can hit about 20mph.

My role on the project was procurement and deal hunting for each part across Facebook Marketplace, Craigslist and eBay.

See BOM.md for the full parts list and photos.md for build photos.
